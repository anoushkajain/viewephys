Release Notes
=============

1.2.0 (2026-04-30)
-------------------

**Added**

- Jump-to-time entry in the navigation window for jumping to a specific
  timepoint without snapping to the slider's fixed chunk boundaries
- Zoom range is now preserved across slider moves and jump-to requests

1.1.2 (2026-04-21)
-------------------

**Added**

- Automatic time axis label in the viewer

**Changed**

- Removed the ``easyqc`` dependency; relevant code is now bundled directly

**Fixed**

- Scrollbar behaviour
- Visibility of viewer menus

1.1.1 (2026-02-16)
-------------------

**Added**

- Option for unfiltered (raw) data and LFP-filtered (broadband) data
  in the binary file viewer
- Wiggle display option

1.0.1 (2024-12-06)
-------------------

**Added**

- Minimal support for Open Ephys files

1.0.0 (2024-10-09)
-------------------

**Added**

- Spike groups in the picking interface
- Live recording option in the binary file viewer
